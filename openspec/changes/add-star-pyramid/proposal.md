## Why

目前專案缺少一個可重複使用、可測試的「印出星星金字塔」範例功能，導致在示範迴圈/字串格式化/輸入參數處理時，需要臨時撰寫零散程式碼。加入此功能可作為教學與驗證輸出的基礎能力，並方便後續擴充（不同高度、對齊方式、填充字元等）。

## What Changes

- 新增可依輸入高度 \(n\) 產生並輸出星星金字塔的功能（每層星號數為 \(2i-1\)，左右以空白置中）。
- 提供一個可重用的產生器/函式，回傳每一行字串（利於單元測試與不同輸出介面重用）。
- 新增 Restful API 帶參數的方式，指定高度並印出結果。

## Capabilities

### New Capabilities

- `star-pyramid`: 依指定高度產生並輸出置中的星星金字塔（可回傳逐行字串，並支援基本參數驗證）。
